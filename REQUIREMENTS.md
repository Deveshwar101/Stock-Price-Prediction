# Requirements

This file lists all the Python packages required to run the Stock Price Predictor App.

## Packages

- **streamlit**: For building the web application.
- **pandas**: For data manipulation.
- **numpy**: For numerical operations.
- **yfinance**: For downloading historical stock data.
- **matplotlib**: For plotting graphs.
- **keras**: For loading the pre-trained model.
- **scikit-learn**: For data preprocessing (e.g., MinMaxScaler).

## Installation

To install all required packages, use:
```bash
pip install -r requirements.txt
